## Description

This repository shows how to implement file upload in nestjs

## Installation

```bash
# clone app to your local machine 
$ git clone https://github.com/JC-Coder/file-upload-nestjs.git

# install needed packages
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
